# RedzSoul-OpenTaiko-Skin
Fork di [Moshir's OpenTaiko Skin](https://github.com/MoshirMoshir/Moshir-OpenTaiko-Skin), aggiornato per supportare **OpenTaiko 0.6.0.107** e personalizzato da RedzSoul.

Vedi [CREDITS.md](CREDITS.md) per i crediti completi.

## How to Install
1. Scarica lo zip cliccando il bottone verde `code` in alto a destra e poi "download zip".
2. Estrai il contenuto nella cartella `System` di OpenTaiko (escludendo README.md e CREDITS.md).
3. Avvia OpenTaiko e seleziona RedzSoul-OpenTaiko-Skin nelle impostazioni.
4. Enjoy!

## FAQ
1. Che versione di OpenTaiko supporta questa skin?
> OpenTaiko 0.6.0.107.
2. Cosa sono i file `.kra`?
> File di lavoro Krita usati per creare i corrispondenti asset `.png`.
3. La skin non funziona!
> Verifica di usare la versione di OpenTaiko supportata. Altrimenti apri una issue su questo repository.
